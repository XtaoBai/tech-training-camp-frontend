# tech-training-camp-frontend


构建的一个所见及所得的 Markdown 编辑器。 目前正在开发中，当前进行输入 markdown 数据语法分析阶段，
尝试自行构建一个抽象语法树解析文档。目标是实现文档云储存和多人编辑。

## 特性

- [x] 基本界面设计
- [x] 工具栏编辑引入
- [x] 编辑区基本功能
- [ ] 编辑区解析与实时预览
- [ ] 用户注册登录与数据库储存
- [ ] 多人协作

## 技术栈
-  Vue 框架
-  iconfont 图标库
-  CodeMirror 支持编辑区编辑
-  Node.js / Flask 支持后端
